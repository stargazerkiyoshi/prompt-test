# 第 6 章 第 0 课：本章导学与主线说明

这一章会把并发工具再往细粒度推进。到这里我们已经知道锁很重要，但也要开始看到：

不是所有并发问题都该靠一把大锁解决。计数、闸门、会合、限流、分阶段等待，经常更适合 CAS、原子类和协调原语。

本章会进入 `AtomicInteger`、CAS、`CountDownLatch`、`CyclicBarrier`、`Semaphore` 等工具，并比较它们和锁的适用边界。

本章课时顺序：

- `lesson-00-chapter-guide.md`
- 第 1 课“CAS 与原子类：什么时候不用锁也能安全更新”
- 第 2 课“`CountDownLatch`、`CyclicBarrier`、`Semaphore` 各管什么”
- 第 3 课“从库存闸门到批处理会合：协调原语的完整例子”
- 第 4 课“本章回收：工具选择不是越高级越好，而是匹配问题”

读完本章后，你应该能回答三个问题：

1. CAS 为什么既高效又有边界。
2. 原子类能解决什么，不能解决什么。
3. 为什么线程协调和线程互斥不是同一件事。
