# 第 1 章 第 2 课：线性组合、张成、线性相关与线性无关

## 先用生活话讲一遍（降抽象预热）

把一组向量想成工具箱里的几件工具。

- 张成在问：这些工具最多能干到什么范围。
- 相关性在问：有没有工具其实是重复的，拿掉也不影响干活能力。

所以这课不是背新词，而是在做“能力边界 + 冗余检查”。


## 先把这节课的问题说透

上一课我们已经知道，向量可以被当作“变化信息包”，而线性组合是把这些信息包按系数组合起来。

但马上会遇到两个更关键的问题：

第一，给定一组向量，它们到底能“造出”多大的结果集合？
第二，这组向量里有没有冗余成员，删掉谁都不影响可造范围？

这两个问题，分别对应今天要建立的两个核心概念：

- 张成（span）：能造出什么
- 线性相关/无关（dependence/independence）：有没有冗余

如果这两个概念不稳，后面你学基、维数、秩都会像背术语；一旦稳住，你会发现后续内容其实都在反复回答“可达范围 + 冗余结构”这对问题。

## 先从上节核心例子继续：配方系统的“可造范围”

我们延续上一课的饮料配方：

- $a = (2, 1, 1)$
- $b = (1, 3, 0)$

任意组合结果写成：$r = x a + y b$。

现在不问某一个具体目标，而是问更大的问题：

当 $x, y$ 任意变化时，所有可能的 $r$ 组成了什么？

这就是“张成”的直觉入口。它不是单个答案，而是“答案全集”。

你可以先这样理解：

- 一个向量给你一条可走方向
- 多个向量给你一个可活动区域
- 张成就是由这些方向拼出来的全部可达点

在二维几何里，这很像“由几根方向杆拼出一条线、一个平面或整个空间”。在我们这个三维营养例子里，$a$ 和 $b$ 通常会张成三维空间中的一个平面（过原点）。

所以你马上能解释上一课的“不可达目标”：它落在这个平面之外，自然就组合不出来。

## 概念关系第一层：线性组合与张成

现在给出必要定义，但仍然贴着问题走。

给定向量组 $v_1, v_2, \ldots, v_k$，所有形如

$$
c_1 v_1 + c_2 v_2 + \cdots + c_k v_k
$$

的结果构成一个集合，这个集合叫做这组向量的张成，记作

$$
\operatorname{span}\{v_1, v_2, \ldots, v_k\}.
$$

请注意这一定义里隐藏的两个关键词：

“所有”与“任意系数”。

只要你把“所有”忘掉，就会把张成误解成“某几个算出来的样例”；只要你把“任意系数”忘掉，就会误以为只允许正权重或整数权重。

从问题角度看，张成回答的是：

这套构件体系，能力边界在哪里。

## 为什么要继续问“冗余”

现在我们故意把系统加一个新向量：

$c = (3, 4, 1)$

并宣称它来自 $a + b$。

你快速验证：

$a + b = (2,1,1) + (1,3,0) = (3,4,1) = c$

于是问题来了：

有了 $c$ 之后，$\operatorname{span}\{a,b,c\}$ 会变大吗？

直觉答案是不变。因为 $c$ 本来就能由 $a,b$ 造出来，它没有带来新方向，只是旧方向的再包装。

这就是“线性相关”要表达的核心事实：

某个向量如果可以由其他向量线性组合出来，它在表达能力上是冗余的。

## 概念关系第二层：相关与无关

现在给最关键的一版定义。

一组向量 $v_1,\ldots,v_k$ 线性相关，指存在一组不全为零的系数 $c_1,\ldots,c_k$，使

$$
c_1 v_1 + \cdots + c_k v_k = 0.
$$

线性无关则相反：只有所有系数全为零时，组合才可能得到零向量。

这一定义初看有点抽象。很多同学会问：

“我明明关心的是冗余，为啥定义写成等于零？”

这是一个非常好的问题。动机是这样的：

- 如果某个向量能由其他向量表示，就能把它移到左边，得到一个“非平凡零组合”
- 反过来，如果有非平凡零组合，也能把某个向量写成其他向量的组合

所以“是否存在非平凡零组合”与“是否有冗余向量”是等价判据。

这个定义厉害的地方在于：它把“冗余”这件事变成了可计算、可证明的统一语言。

## 在例子里把定义真正走完

我们用 $a,b,c$ 来做完整验证。

已知 $c = a + b$，改写成

$a + b - c = 0$

这里系数是 $(1, 1, -1)$，显然不全为零。因此 $a,b,c$ 线性相关。

同时，$\operatorname{span}\{a,b,c\} = \operatorname{span}\{a,b\}$，因为 $c$ 没有提供新方向。

现在换一个对照场景：只看 $a,b$。

要判断 $a,b$ 是否无关，就是看

$x a + y b = 0$

是否只有 $x=0,y=0$ 一个解。

把分量展开：

- $2x + y = 0$
- $x + 3y = 0$
- $x = 0$

第三式给 $x=0$，代回得 $y=0$。只有零解，因此 $a,b$ 线性无关。

你会看到一个很重要的结构事实：

- $a,b$ 无关，说明这两根方向杆都有效
- 加入 $c=a+b$ 后变相关，说明系统出现冗余
- 冗余出现时，张成能力不一定增强

这三个判断，后面会直接升级成“基与维数”的语言。

## 形式化为什么在这里不可再省

到这一步我们可以明确说：

如果只靠图形直觉，在高维数据里你几乎无法可靠判断相关/无关；必须依赖方程化判据。

所以今天出现的形式化不是“数学洁癖”，而是现实需求：

- 你要让判断可算法化
- 你要让结论可复核
- 你要从二维直观跨到高维结构

这也是线代的一条主线：

每当直觉开始失去可操作性，形式化就作为“保持可控”的工具出现。

## 本节局部闭环：我们现在到底掌握了什么

把今天的学习收束成一条完整链路：

我们先问“这组向量能做什么”，得到张成；再问“这里面有没有重复劳动力”，得到相关/无关；然后用零组合判据把冗余问题转成可计算问题，并在同一个配方案例里走完了“无关到相关”的转变。

如果压缩成一句工作语言：

张成讲能力边界，相关性讲能力是否被冗余表达。
