# 02 盘古：世界骨架初始化

## 1. 本轮世界展开思路

本轮不把混沌层的六条法则逐条翻译成地理名词，而是先回答一个更根本的问题：

**当创生、灵能、承载、循环、平衡与回归同时长期存在时，世界为什么会长成现在这样的空间结构。**

因此，本轮重点不是罗列奇观，而是先建立世界的整体生成逻辑：

1. 原初之力不会平均落在世界各处。
2. 力量载体也不会平均分布。
3. 灵能流动会沿着更容易承载、释放和回收的路径形成长期走向。
4. 当力量在某些地方过度聚集时，平衡会把压力转移出去。
5. 被耗尽、衰败、未完全消散的东西，最终会沉到更容易回归的地带。

也就是说，这个世界的地理骨架不是“先有几块地，再给它们加属性”，而是：

**原初之力的涌出 + 承载结构的锁定 + 灵能流动的迁移 + 平衡的再分配 + 回归的沉降**

共同塑造出来的结果。

## 2. 总体世界骨架

测试世界 02 的客观世界可暂定为一个“中部稳定、边缘活化、低地沉降、力量沿骨架流动”的陆地世界。

它的基本形态不是均匀铺开的平面大陆，而更像一个长期处于生长与回落之间的活体结构：

1. 世界边缘更接近原初之力
   边缘地区更容易出现新生地貌、断裂海岸、异变环境和未完全稳定的土地。这里不是因为“被设定成危险”，而是因为创生力量最难被完全锁住。

2. 世界中部存在稳定承载骨架
   古老岩层、高地、深矿脉、巨型林带和地脉节点在长期循环中逐渐成为最能稳定承受力量的区域。它们构成了世界的硬骨架。

3. 灵能沿骨架与低地之间形成长期流路
   灵能不会乱流，而会顺着地势、载体质量和环境惯性形成长期主流方向。于是中部平野、河谷和缓丘成为最适合生命繁盛的地带。

4. 世界内部会自然形成回归沉积区
   凡是被耗尽、衰败、压退、埋藏或未完全消散的东西，最终都会向更低、更闭塞、更难完全更新的区域沉积。于是旧时代残留、残响、遗迹和衰败生态会在少数区域长期堆积。

因此，这个世界的宏观结构可以概括为：

**外缘不断生长，中部持续承载，平野负责流转，低地负责回收。**

## 3. 世界为何会形成这样的结构

为了让后续模块能够承接，本轮直接写明生成关系。

### 3.1 创生与承载共同制造“边缘活化，中部稳固”

原初之力总在尝试生成新的物质、生命和结构，但真正能长期留下来的部分，必须找到足够强的载体。

于是结果不是“整个世界都同样活跃”，而是：

- 靠近原初之力的边缘地带不断长出新东西，但不够稳定。
- 能承受力量的中部高地、古岩层和深层节点，反而会慢慢成为长期稳固的世界骨架。

所以边缘更新快，中部更稳。

### 3.2 灵能与循环共同制造“流动平野”

当力量被中部骨架锁住后，它不会停在那里，而会沿着地势、河道、风带、林带和地脉持续流出。

于是世界不会只剩下“危险边缘”和“死硬高地”，中间必然会长出大片流动而丰饶的生命地带：

- 灵能经过的地方更适合生命繁盛。
- 生命繁盛后又会反过来参与循环，进一步稳定这些地带。
- 文明未来也更容易在这里出现，因为资源、人口和路径都会向这里汇聚。

所以中域平野不是装饰，而是法则共同作用后最自然出现的生命带。

### 3.3 平衡与回归共同制造“沉降低地”

力量不会只向上累积，也不会永远停在最强的地方。

当灵能、生命、文明或旧时代结构被过度消耗、压制或推迟终结时，世界会通过回归把它们送去更容易沉积的地方。

于是低地、盆地、旧河床、塌陷区和被遗弃的旧环境会不断吸纳：

- 衰亡生命
- 旧时代遗迹
- 失衡后果
- 未完全回归的力量
- 难以消散的残响

所以回归地带不是“额外加的废墟区”，而是这个世界维持长期平衡所必须拥有的回收层。

## 4. 四个关键区域

### 区域一：裂岸外缘

#### 基本环境特征

裂岸外缘位于世界边缘，是原初之力最容易直接影响现世的地带。这里由断裂海岸、破碎岬角、雾岛、塌陷浅海和新生岩地组成。

它的核心特征不是单纯危险，而是：

**新东西总在这里长出来，但很少能立刻稳定下来。**

#### 为什么会形成

因为这里更接近原初之力，但缺少足够稳定的大型载体，所以创生持续发生，定形却总是不完全。

#### 对后续模块的约束

- 女娲在这里生成生命时，应优先考虑高变异、高恢复、高环境感知或强迁徙能力。
- 伏羲若要在这里生成文明，应该偏边境、港口、探采、禁区和短期驻留结构，而不是长期稳定帝国核心。

### 区域二：灵流平野

#### 基本环境特征

灵流平野位于世界中部偏低地带，由河网、缓丘、大平原、森林边缘和周期性湿地共同组成。

它的核心特征是：

**力量会经过这里，生命会在这里放大。**

#### 为什么会形成

因为中部骨架锁住了大部分高强度力量，而外泄出来的灵能沿着较容易流通的地势和环境结构形成长期流路，于是中间必然出现一片适合流转和繁盛的区域。

#### 对后续模块的约束

- 女娲应把这里视为主要生命繁盛带，而不是单一物种独占区。
- 伏羲未来若生成大规模文明、农业、商路和人口中心，这里应是首选承接区。

### 区域三：承石高地

#### 基本环境特征

承石高地是世界中部偏高处的稳定高地系统，由高山、石台、深矿层、古树根系带和大型地脉节点构成。

它的核心特征是：

**力量在这里不一定最活，但最容易被保存。**

#### 为什么会形成

因为承载法则会在长期循环中筛选出最适合稳定力量的物质与结构。久而久之，这些区域自然成为世界的硬骨架和高承载中心。

#### 对后续模块的约束

- 女娲若生成长寿种、强血脉种或依赖器物/矿脉的生命，这里是高优先承接区。
- 伏羲未来若生成圣地、王权中心、法师塔、矿脉争夺或古老宗教中枢，这里应是主舞台之一。

### 区域四：归墟盆地

#### 基本环境特征

归墟盆地位于内陆低地和封闭沉降区，由塌陷平原、旧河道、灰林、沉降湖、废弃地层和深埋遗迹组成。

它的核心特征是：

**世界不愿立刻放下的东西，最后都会慢慢堆到这里。**

#### 为什么会形成

因为平衡和回归不会让所有后果即时消失，很多被推迟终结的力量、结构和生命会沿着沉降路径进入更低、更闭塞的地带，逐渐累成残响和旧时代沉积。

#### 对后续模块的约束

- 女娲在这里生成生命时，应优先考虑低活性、耐残响、慢周期或依附旧遗留物生存的生命。
- 伏羲未来若生成遗迹争夺、禁地社会、旧王朝残民或祭祀型聚落，这里会很重要。

## 5. 潜在模糊点或待定问题

1. 世界整体尺度仍未确定：当前只确定骨架关系，未确定单大陆、多大陆还是大陆群结构。
2. 四个关键区域的精确方位仍未固定：本轮确定的是生成关系，不是地图坐标。
3. 承石高地与灵流平野之间是否存在清晰的过渡带，目前未定。
4. 承石高地的高承载结构主要偏矿脉、古树、岩层还是人工巨构，目前未定。
5. 归墟盆地中的残响更偏资源型、污染型还是混合型，目前未定。
6. 灵能流动是否存在一两条绝对主流路，还是以多支流并行结构为主，目前未定。
7. 哪些区域最适合作为女娲层的第一批生命起源地，仍待后续决定。
8. 奇迹法则更容易在裂岸外缘还是归墟盆地显现，仍待后续决定。

## 6. 本轮更新摘要

本轮完成测试世界 02 的盘古层重写。

已确定总体世界骨架：

- 世界不是“几种法则对应几种地形”，而是由法则共同作用生成的整体结构。
- 原初之力更容易活化世界边缘。
- 高承载结构更容易稳固世界中部。
- 灵能流动更容易在中域形成生命繁盛带。
- 平衡与回归更容易在低地形成残响沉积区。

已生成四个关键区域：

1. 裂岸外缘
   更接近原初之力；新生快；稳定差；适合异变与边境探索。

2. 灵流平野
   灵能长期经过；生命繁盛；适合生态扩张与未来文明中心。

3. 承石高地
   高承载结构集中；适合保存力量、形成圣地、矿脉和权力中枢。

4. 归墟盆地
   回归与残响沉积最深；适合遗迹、禁地、旧时代残留和慢周期生态。

对后续模块的主要约束：

- 女娲需要基于“边缘活化、中域繁盛、高地承载、低地回归”的差异生成生命。
- 伏羲需要把资源、圣地、权力中心、禁区和迁徙路径建立在这四种空间功能差异之上。
- 后续扩展不能把这些区域当成互不相干的景观点，而要保持它们来自同一套世界生成逻辑。
