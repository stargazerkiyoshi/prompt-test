# 测试世界 01：世界状态卡

## 1. 世界基础信息

- 世界名称：测试世界 01
- 当前版本：v0.4-fuxi
- 当前阶段：创世
- 当前总体风格：理性世界构造；周期性兴衰；环境、生命与文明受底层能量机制约束
- 当前世界一句话概述：一个受周期性潮能影响，并由动态区域、两类智慧生命与三种周期适应型社会共同构成的世界。

## 3. 当前稳定层

- 世界底层核心机制是周期性能量潮汐。
- 能量潮汐存在涨潮、满潮、退潮、枯潮与回潮阶段。
- 世界变化具有相位差，不同层级对潮汐响应不同步。
- 原初能量遵守转化与代价原则，不允许无成本无限使用。
- 所有存在形式都必须有适应窗口，不能全阶段绝对优势。
- 结构会在关键阈值处发生重组，世界历史天然包含兴衰、残留和再组织。
- 已确定三种原初能量形式：潮势、潮核、潮痕。
- 世界骨架由外层潮动带、中层储核带、内层滞痕带构成。
- 已确定三个关键区域：外潮海陆带、沉核山原、旧痕盆地。
- 三个关键区域对潮能响应存在相位差：外潮海陆带最快，沉核山原居中，旧痕盆地最慢。
- 已确定两类主要智慧生命：洄行型智慧生命、潜衡型智慧生命。
- 洄行型智慧生命主要适应外潮海陆带和外层潮动带，依靠迁徙、分群和阶段性繁衍跨越周期。
- 潜衡型智慧生命主要适应沉核山原、旧痕盆地及其交界区，依靠储能、低代谢、深层避险和长期记忆跨越周期。
- 已确定三种主要社会形态：洄路群社会、沉库据点社会、潮界互换社会。
- 洄路群社会以迁徙路线、时段协调和分群重聚为秩序基础。
- 沉库据点社会以储备管理、避险空间和长期记忆为秩序基础。
- 潮界互换社会以时段交换、风险分担、信息校验和接触节点为秩序基础。

## 4. 当前活跃层

- 下一轮最适合进入羲和模块，基于三种社会形态生成时代阶段与长期压力。
- 羲和应优先处理周期变化如何推动洄路群社会、沉库据点社会和潮界互换社会的兴衰、收缩、协作和重组。
- 当前仍不宜直接生成具体人物或完整事件链。

## 5. 待解决问题

1. 一个完整潮汐周期持续多久，是否存在多重周期嵌套。
2. 潮汐来源是否需要解释，或保持为不可追溯底层法则。
3. 不同区域是否同步涨落，还是存在显著相位差。
4. 超凡机制是否存在；若存在，其具体使用边界和代价是什么。
5. 智慧生命或文明是否能够预测潮汐周期。
6. 潮核资源是否稀缺，以及其衰减、污染或泄露形式如何表现。
7. 潮痕对地理、生命和文明惯性的影响范围需要后续分层细化。
8. 世界是单一大陆、多大陆，还是陆海群岛结构。
9. 三个关键区域之间的精确方位、面积比例和边界线。
10. 沉核山原中的潮核主要依附于矿物、地下水、晶质岩层还是其他载体。
11. 旧痕盆地保留的是偏资源型残余，还是偏危险型残余。
12. 两类智慧生命是否存在共同祖源。
13. 两类智慧生命是否可以稳定杂居、竞争或形成互补关系。
14. 两类智慧生命的具体繁衍结构和幼体生存条件。
15. 生命体对潮能的感知是主动感知、被动感知，还是环境信号间接判断。
16. 潜衡型智慧生命的储能组织是否直接储存潮能。
17. 洄行型智慧生命的迁徙路线是否跨越全部三类区域。
18. 非智慧生态生命的食物链、共生链和竞争链基础。
19. 洄路群社会的路线知识如何记录和传承。
20. 沉库据点社会的潮核管理是否形成专门阶层。
21. 潮界互换社会是否具有独立身份，还是只是两类社会的接触机制。
22. 三种社会形态之间是否存在贸易、冲突、依附或联盟关系。
23. 是否存在跨社会的通用计时、潮相判断或资源度量方式。
24. 潮能预测能力是否可靠，以及不同社会是否拥有不同预测精度。
25. 社会内部的亲属结构、繁衍安排、教育方式和权威继承规则。
